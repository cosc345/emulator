Emulator Description

The emulator is use to emulate a very simple smart watch. It does have any sophisticated features of a real smart watch and does not mean to emulate a smart watch of any kind. It only purpose is to run a simple prototype health application. The emulator contain a basic canvas element and a smart watch image background. It contain a simple load and clear buttons for loading and clearing apps. It also contain a monitor, which is use to interpret the effects and problems of the app.

Features

Size: 300 x 300
Languages used: JavaScript and HTML5
Other essential packages: jQuery version 1.11 from Google's Ajax library

Emulator Installation

You can download our emulator from github: Download

Navigation

1. To navigate and run the emulator, click the emulator file.
2. Click on public_html file.
3. Then click setup.html

Running the emulator

To be able to run the emulator, just click the setup.html file
But first you need to install a server using python and here's how

Server Installation

1. If you don't have python installed. Download the latest version of python
2. Under command line environment, navigate to the file that has the setup.html in it.
3. Type in this command -m SimpleHTTPServer 8080.

Running the Application

For users

For the most part it is difficult to install a new app without prior experience with HTML and javascript.
But for the pre-installed apps, you can select any app from the selection tray and click load
To clear app, click clear

For Developers

For running the pre-installed apps, the instruction is much the same as the users instruction.
For new apps, follow this instruction closely.

1. The app must be in Javascript.
2. You can put the app file anywhere you like. But it must be within the public.html file.
3. Use the path for the application to run the app.
4. Add your application path finder with the selection tray of the setup.html.
5. The value attribute of the app's selection must be your source path. For example HealthApp/health.js.

To Run the application

The canvas name is 'emulatorCanvas'.
Use the runningScript(file) method under emulatorForm.js to run the app.
